# Advent of Code 2018

## Clojure

Dependencies:

 * Clojure binary https://clojure.org/guides/getting_started

Run:

```
cd in_clojure
make
```

## Go

Dependencies:

 * Go https://golang.org/doc/install

Run:

```
go get -u git@github.com:Jell/advent2018.git
cd in_go
make
```
