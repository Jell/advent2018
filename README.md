# Advent of Code 2018

## Haskell

Dependencies:

 * Stack https://docs.haskellstack.org/en/stable/README/

Run:

```
cd haskell
make
```

## TypeScript

Dependencies:

 * Yarn https://yarnpkg.com/en/

Run:

```
cd typescript
yarn install
make
```

## Go

Dependencies:

 * Go https://golang.org/doc/install

Run:

```
go get -u git@github.com:Jell/advent2018.git
cd go
make
```
